Odoo Easy Signin Signout

This module allow user to signin signout easily

We are happy to solve issue so please contact if any issue you find. In case of suggession also contact us. Contact for free support.

ScreenShot and Flow of Module

When employee not login in system it will shows green login icon.

For Signin user need to click on green button.

When user click on signin button it will popup successfully signin.

After user signin icon converted to red color.

When employee login in system it will shows red signout icon.

For Signout user need to click on red button.

When user click on signout button it shows popup successfully signout.

After user signout, Icon converted to green color again.

Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
